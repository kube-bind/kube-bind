    bind

    Synopsis

    kube-bind allows you to bind remote services into your cluster using either a web UI or command-line interface.

    By default, 'kubectl bind' opens the kube-bind web UI in your browser.

    For more information, see: https://docs.kube-bind.io

    bind [server-url] [flags]

    Examples

        # kubectl bind login --server https://my-kube-bind-server.com

    # Open kube-bind UI for current server context 
    kubectl bind

    # List available templates (CLI mode)
    kubectl bind templates

    # Bind specific template (CLI mode)
    kubectl bind apiservice --name my-api --template-name database-service

    # View template details without binding
    kubectl bind apiservice --name my-api --template-name database-service --dry-run
